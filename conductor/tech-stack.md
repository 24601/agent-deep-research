# Tech Stack

## Programming Language
- **Language:** TypeScript
- **Runtime:** Node.js

## Core Libraries
- **CLI Framework:** None (Integration as a Gemini CLI Extension)
- **AI SDK:** `@google/genai` (Official SDK)
- **Validation:** `zod`
- **Protocol:** Official MCP (Model Context Protocol) libraries for extension integration.

## Development Tools
- **Linter/Formatter:** ESLint, Prettier
- **Build Tool:** `tsc` (TypeScript Compiler)
